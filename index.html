This is a a helping website/app, which has the following three functions to help a reader:

    1. Position check is meant to inform the user if his position is right for reading, so that he does not hurt his back and neck.
    2. Light sensor is used to determine if the user is reading in a dark environment and the color of the background is adjusted accordingly.

    Warning: Light sensors are only detectable on Firefox browsers!

    3. Reading time is counted and the user gets a notification/alert when he must do a break, so that his eyes do not get tired.
